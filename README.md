# AI Chatbot

A minimal, beautiful AI chatbot powered by **Google AI Studio** (Gemini).

## Setup

```bash
npm install
```

Your API key is in `.env`. For production, use environment variables and never commit real keys.

## Run

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000).

> **⚠️ Port conflicts:** If you see an `EADDRINUSE` error when
> starting the server, port 3000 is already in use. Stop the process
> using that port or start the app with a different port, e.g.
> `PORT=4000 npm start`.
